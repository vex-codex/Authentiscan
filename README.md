# 🔍 AuthentiScan

AuthentiScan is an AI-powered web application designed to verify product authenticity and detect fake or misleading reviews. It helps users make informed purchasing decisions by analyzing products and customer feedback.

## Features

- Product authenticity verification
- Fake review detection
- AI-powered analysis
- User-friendly interface
- Secure login system
- FastAPI backend
- Responsive frontend

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- FastAPI
- Python

## Installation

Clone the repository

```bash
git clone https://github.com/vex-codex/Authentiscan.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the server

```bash
python server.py
```

Open

```
http://127.0.0.1:8000
```
